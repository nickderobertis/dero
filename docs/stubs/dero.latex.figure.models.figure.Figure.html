dero.latex.figure.models.figure.Figure¶

class dero.latex.figure.models.figure.Figure(subfigures: typing.List[typing.Union[dero.latex.figure.models.subfigure.Subfigure, dero.latex.figure.models.graphic.Graphic]], caption=None, label=None, centering=True, position_str=None, landscape: bool = False)[source]¶

used for creating latex figures from images. Currently the main usage is the Figure class created with the method Figure.from_dict_of_names_and_filepaths. Pass a dictionary where the keys are names for subfigures and the values are filepaths where the image for the subfigure is located.
